Understanding Python Decorators

17 March 2026
1 min read
#python #programming #decorators #functions #tutorial

Python decorators are a powerful tool that allow you to modify the behavior of a function or class without changing its source code.

In simple terms, a decorator is a function that takes another function as an argument, adds some functionality, and returns a new function. This can help make your code more modular, reusable, and readable.

Here's a basic example:

def decorator(func):
    def wrapper():
        print("Before the function runs")
        func()
        print("After the function runs")
    return wrapper

@decorator
def say_hello():
    print("Hello!")

say_hello()

The @decorator syntax is a shorthand for say_hello = decorator(say_hello). This makes it easy to apply the same decorator to multiple functions.

Decorators can also take arguments, be stacked, and be used with classes. They are commonly used for logging, authentication, caching, and more.

Understanding decorators can greatly improve your ability to write clean and professional Python code.
