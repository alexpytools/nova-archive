Mastering Python Generators

17 March 2026
1 min read
#python #programming #generators #memory #tutorial

Python generators are a type of iterable, like lists or tuples, but they generate items on the fly instead of storing them all in memory. This makes them perfect for working with large datasets or streams of data.

You create a generator using a function with the yield keyword:

def my_generator(n):
    for i in range(n):
        yield i*i

gen = my_generator(5)
for value in gen:
    print(value)

Output:

0
1
4
9
16

Notice that the values are produced one at a time and not stored all together. This is memory-efficient and allows processing very large datasets without using a lot of RAM.

You can also use generator expressions, similar to list comprehensions:

squares = (x*x for x in range(10))
for s in squares:
    print(s)

Generators are widely used in Python for file reading, data pipelines, and lazy computations. Learning to use them effectively can make your Python programs faster, more efficient, and more scalable.
